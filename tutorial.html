Build Guide · LineageOS 18.1

Samsung Galaxy Grand 2 Duos

Kompilasi LineageOS 18.1 dari sumber untuk SM-G7102 — codename ms013g. Rantai dependensi, branch, dan langkah build yang sudah diverifikasi dari organisasi Grand-2-Rebirth.

SoC Snapdragon 400 · MSM8226 RAM 1.5 GB CPU Quad Cortex-A7 1.4 GHz lunch lineage_ms013g
⚠ Temuan — soal branch "Go"

Perbandingan langsung lineage-18.1-Go vs lineage-18.1 pada common tree: branch “Go” itu snapshot lama (ahead 0, behind 15). Branch lineage-18.1 punya 15 commit tambahan — fix RIL/telephony, Bluetooth, brightness, dan koreksi baterai 2600 mAh.

Seluruh device tree juga di-grep: tidak ada ro.config.low_ram=true maupun go_defaults.mk — jadi ini bukan Android Go dalam arti teknis, hanya penamaan.

Rekomendasi: pakai lineage-18.1 untuk common tree. Kalau ingin perilaku Go/low-RAM asli (masuk akal untuk 1.5 GB), aktifkan lewat ENABLE_GO=1 ./build.sh — lihat blok Go di bawah.

Tool repo, git, dan ccache

repo + ccache setup
# pasang tool repo
mkdir -p ~/bin
curl https://storage.googleapis.com/git-repo-downloads/repo > ~/bin/repo
chmod a+x ~/bin/repo
echo 'export PATH=~/bin:$PATH' >> ~/.bashrc

# identitas git + git-lfs
git config --global user.email "ryanbagas27@gmail.com"
git config --global user.name  "Ryan"
git lfs install

# ccache — percepat build ulang
echo 'export USE_CCACHE=1' >> ~/.bashrc
echo 'export CCACHE_EXEC=$(which ccache)' >> ~/.bashrc
source ~/.bashrc
ccache -M 30G
2

Inisialisasi source LineageOS 18.1

repo init
mkdir -p ~/android/lineage && cd ~/android/lineage
repo init -u https://github.com/LineageOS/android.git -b lineage-18.1 --git-lfs

Tip: tambahkan --depth=1 untuk hemat disk/bandwidth (shallow clone).

3

Pasang local_manifest

Salin file ms013g.xml (sudah disediakan) ke tempat yang benar:

local_manifests/ms013g.xml
mkdir -p ~/android/lineage/.repo/local_manifests
cp ms013g.xml ~/android/lineage/.repo/local_manifests/ms013g.xml

Manifest ini pakai dua remote: rebirth (→ Grand-2-Rebirth) untuk semua repo device/kernel/vendor, dan los (→ LineageOS) khusus untuk hardware/sony/timekeep.

4

Sync source

repo sync
cd ~/android/lineage
repo sync -c -j$(nproc --all) --force-sync --no-clone-bundle --no-tags

# verifikasi dependensi kunci ikut ke-clone
ls device/samsung/ms013g device/samsung/ms01-common \
   kernel/samsung/msm8226 vendor/samsung/ms013g hardware/sony/timekeep

Ukuran: sekali sync ± 40–60 GB. Sabar di koneksi lambat.

5

Build

envsetup · lunch · mka
cd ~/android/lineage
source build/envsetup.sh

# set target (breakfast) lalu lunch
breakfast ms013g
lunch lineage_ms013g-userdebug

# bikin zip installable
mka bacon -j$(nproc --all)

Hasil ada di out/target/product/ms013g/:

  • lineage-18.1-<tanggal>-UNOFFICIAL-ms013g.zip — paket yang di-flash
  • recovery.img, boot.img
Opsi — aktifkan Android Go / low-RAM

Untuk perilaku Go asli di RAM 1.5 GB, tambahkan properti low-RAM ke device/samsung/ms013g/system.prop (skrip build.sh melakukannya lewat ENABLE_GO=1):

system.prop — tambahan
ro.config.low_ram=true
ro.lmk.use_minfree_levels=true
dalvik.vm.heapgrowthlimit=96m
dalvik.vm.heapsize=256m
pm.dexopt.first-boot=quicken
🔐

microG (opsional)

Integrasi lineageos4microg memakai repo modern vendor/partner_gms. Aktivasi lewat WITH_MICROG=1 menjalankan tiga langkah otomatis:

  • Unduh APK — GmsCore, GsfProxy, FakeStore, F-Droid + Privileged Extension. APK tidak disimpan di git; diunduh saat build oleh vendorsetup.sh → butuh koneksi internet.
  • Sertakan paket — inherit vendor/partner_gms/products/gms.mk ke lineage_ms013g.mk.
build dengan microG
# microG + optimasi Go sekaligus (rekomendasi untuk 1.5 GB RAM)
WITH_MICROG=1 ENABLE_GO=1 ./build.sh build
Signature spoofing — sudah bawaan LOS 18.1

LineageOS 18.1 sudah punya signature spoofing restricted di frameworks/base (MICROG_REAL_SIGNATURE + isMicrogSigned() di PackageManagerService). Hanya microG asli (yang PRESIGNED) yang boleh menyamar jadi signature Google. Jadi tanpa patch pun microG jalan.

AspekRestricted (default)Unrestricted (opsional)

Patch frameworks/basetidakya

Spoofing generik (jarang perlu, melemahkan keamanan) diaktifkan terpisah:

opsional — unrestricted
# patch frameworks/base -> semua app boleh spoof (opsional)
WITH_MICROG=1 MICROG_UNRESTRICTED=1 ./build.sh build
Verifikasi setelah flash

Buka app microG Settings → Self-Check. Baris “System grants signature spoofing” harus tercentang ✓. Lalu login akun Google & aktifkan Cloud Messaging untuk push notification.

6

Flash ke perangkat

  • Unlock bootloader & pasang recovery (TWRP / Lineage Recovery). Grand 2 pakai Odin di mode Download (Vol Down + Home + Power).
  • Boot ke recovery → Wipe system, data, cache, dalvik.
adb sideload
adb sideload lineage-18.1-*-UNOFFICIAL-ms013g.zip

GApps opsional — untuk Go + 1.5 GB RAM, pertimbangkan varian ringan (NikGApps “go”) atau tanpa GApps. Boot pertama 5–10 menit.
